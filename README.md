# SuroClima Instalaciones

Página web oficial de SuroClima Instalaciones — expertos en sistemas de climatización, aerotermia, calefacción y fontanería en Badajoz.

🌐 **Web en producción:** [Ver en Vercel](https://suroclima-badajoz.vercel.app)

## Tecnologías
- HTML5 + Tailwind CSS (CDN)
- Google Fonts (Inter, Montserrat, Material Symbols)
- Sin dependencias de build — despliegue estático directo

## Despliegue
Conectado a Vercel vía GitHub. Cada push a `main` despliega automáticamente.
